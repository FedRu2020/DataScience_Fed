основные цели и задачи проекта
	
	провести разведывательный анализ данных и составить отчёт по его результатам для построения модели, которая 
	предсказывала бы результаты госэкзамена по математике для каждого ученика школы 

	
краткая информация о данных
	датасет из 30 столбцов и 395 строк.
	всего в датасете 13 числовых и 17 строковых столбцов.
	Только первые три столбца не содержат пустых значений.


этапы работы над проектом
	1. Первичный осмотр данных.
	2. Анализ данных в каждом столбце в следующем порядке:
		- Осмотр данных.
		- Проверка данных на пустые значения.
		- Проверка данных на дублированные/полностью скоррелированные значения.
		- Проверка данных  на наличие выбросов.
		- Отбор данных, пригодных для дальнейшего построения модели.
	3. Корреляционный анализ данных
		Проводился для 12 числовых столбцов.
		Числовой столбец studytime, granular исключен из анализа так как по значениям дублирует studytime.
	
	4. Анализ номинативных переменных
		Проводился для всех 17 строковых столбцов.
